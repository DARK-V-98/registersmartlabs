# **App Name**: SimpleCal

## Core Features:

- Calendar Display: Display a full-sized, interactive calendar.
- Date Navigation: Enable users to navigate between months and years using intuitive controls.
- Event Highlighting: Visually highlight days with events.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a calm, focused feel.
- Background color: Very light Indigo (#E8EAF6), almost white, with a hint of the primary hue.
- Accent color: Muted Violet (#9575CD), a color harmonious with Indigo but noticeably different in brightness.
- Font: 'Inter' (sans-serif) for a clean and modern look.
- Simple, single-column layout to showcase the calendar prominently.
- Minimalist icons for navigation (e.g., arrows) and possibly event categories.
- Subtle transition animations when switching between months.
